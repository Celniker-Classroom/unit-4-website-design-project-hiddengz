Titan Attack is a game that I made in may of 2025 back when i first used github. It includes only one titan. the name is missleading because I thought it sounded cool when I was 13.

I was taking a coding class at the time to learn python which is the main reason I was using github in the first place.

About 90% of the game was vibecoded and I only spend a week or two on it. there used to be a jungle boss in the jungle area but i lost the code.

The game includes heavy references to minecraft dungeons with the design of the jungle area along with the music being from it
